# Exercicis — 4t de Primària

Aplicació web d'exercicis interactius per a 4t de primària. Sense instal·lació, sense servidor — funciona directament al navegador.

**🌐 [Obrir app](https://D4rumanDev.github.io/exercicis-primaria/)**

## Matèries

- **Matemàtiques** — Suma, resta, multiplicació, divisió, taules, problemes, fraccions, diners, hores, mesures, geometria, estadística
- **Llengua** — Comprensió lectora (20 textos), morfologia, mots encreuats, cartells, sopa de lletres
- **Anglès** — Horari, professions, activitats, la ciutat, descripció, word search, crossword
- **Castellano** — Ortografía (B/V, J/G, LL/Y, R/RR), vocabulario, familias de palabras, sinónimos, signos de puntuación, comprensión lectora (20 cuentos), sopa de letras, crucigrama

## Característiques

- Sistema de gamificació amb XP, nivells, ratxes i assoliments
- Dificultat adaptativa (algorisme ZPD) als exercicis de matemàtiques
- Estadístiques detallades per matèria amb gràfic dels últims 30 dies
- Mode fosc automàtic (segueix el sistema, amb toggle manual)
- Responsive — funciona en mòbil i tauleta
- Tot el progrés es desa localment al navegador
